🧠 Expert System in Python

A simple rule-based expert system implemented in Python using forward chaining.

🚀 Overview

This project demonstrates how an expert system works using:

* Rule-based logic
* Forward chaining inference
* Problem detection & solution suggestion

⚙️ Features

* Takes user input as symptoms/problems
* Applies rules to derive conclusions
* Detects problems automatically
* Suggests solutions
* Displays reasoning steps (explainable AI)

🧩 How It Works

1. User enters input (facts)
2. System checks rules (IF-THEN)
3. New facts are derived
4. Final output is generated

📌 Example

Input:engine not starting, battery low

Output: Detected Problems: battery issue
        Suggested Solutions: replace battery

🛠️ Technologies Used : Python 3


▶️ How to Run

1. Clone the repository:

git clone https://github.com/your-username/expert-system-python.git

2. Navigate to folder:

cd expert-system

3. Run the program:

python expert_system.py
